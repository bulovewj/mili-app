## MVP — v0.1 : 뼈대와 화면 흐름

**목표**: 앱의 전체 구조가 작동하는 껍데기 완성. 디자인과 화면 이동이 되면 성공.

### 포함 기능
- [ ] index.html 단일 파일 구조 셋업
- [ ] 전체 화면 라우팅 시스템 (JS로 화면 전환)
- [ ] Screen 1: 시작 화면 (이름 입력)
- [ ] Screen 2: 앱 소개 화면
- [ ] Screen 5: 홈 런처 (더미 카드 5개, 클릭 안 됨)
- [ ] localStorage 기본 구조 (이름 저장/불러오기)
- [ ] 모바일 퍼스트 레이아웃 (390px 기준)
- [ ] 전체 디자인 시스템 (컬러, 폰트, 버튼 스타일)
- [ ] 화면 전환 애니메이션 (페이드)

### 완료 기준
> 이름 입력 → 소개 화면 → 홈 런처까지 이동이 되고, 디자인이 귀엽고 모바일에서 잘 보이면 MVP 완료.

---

## v0.2 : 진단 퀴즈 + 5개 시뮬레이션 콘텐츠

**목표**: AI 없이도 모든 화면을 처음부터 끝까지 체험할 수 있는 상태.

### 포함 기능
- [ ] Screen 3: Module 01 진단 퀴즈 (4문항 객관식)
- [ ] Screen 4: 진단 결과 화면 (자동 채점, 등급 표시)
- [ ] Screen 5: 홈 런처 진행률 연동 (완료 카드 체크 표시)
- [ ] Screen 6: 유튜브 탐정 (유튜브 UI 재현 + 서술형 입력)
- [ ] Screen 7: 뉴스 비교 (뉴스 앱 UI 재현 + 서술형 입력)
- [ ] Screen 8: 광고 탐정 (SNS 피드 UI 재현 + 서술형 입력)
- [ ] Screen 9: AI 탐정 (카카오톡 UI 재현 + 서술형 입력)
- [ ] Screen 10: 카드뉴스 만들기 (편집 UI + 입력)
- [ ] Screen 11: AI 채점 결과 화면 (임시 더미 결과 표시)
- [ ] Screen 12: 역량 지형도 (Canvas/SVG 레이더 차트, 더미 데이터)

### 완료 기준
> API 키 없이도 모든 화면을 처음부터 끝까지 탭으로 이동하며 체험 가능. 더미 점수로 역량 지형도까지 표시되면 완료.

## v0.3 : Claude API 연동

**목표**: 실제 AI 채점과 피드백이 작동하는 상태.

### 포함 기능
- [ ] Claude API 키 설정 구조 (Netlify 환경변수 `ANTHROPIC_API_KEY`)
- [ ] `security.md` 기준 서버리스 함수 프록시 구조 적용
- [ ] 활동별 루브릭 시스템 프롬프트 완성 (5종)
- [ ] Screen 11: 실제 AI 루브릭 채점 연동
  - 등급(상/중/하) + 점수(85/65/50) + 피드백 + 심화질문 표시
- [ ] Screen 12: 종합 피드백 AI 생성 연동
- [ ] API 호출 로딩 UI (로딩 스피너/애니메이션)
- [ ] API 실패 에러 처리 (재시도 1회 → 안내 메시지)
- [ ] 안전 필터링 확인 (욕설/개인정보 입력 시 거부 안내)
- [ ] localStorage에 AI 채점 결과 저장

### 완료 기준
> 실제 서술형 답변 입력 후 AI가 상/중/하 등급과 피드백을 제대로 반환하면 완료. 5개 활동 모두 채점 테스트 통과.

## v0.4 : 품질 개선 및 엣지케이스 처리

**목표**: 실제 수업에서 써도 문제없는 안정성 확보.

### 포함 기능
- [ ] 입력값 유효성 검사 (최소 글자 수 미달 시 안내)
- [ ] 앱 껐다 켜도 이어하기 (localStorage 복원 로직)
- [ ] 이미 완료한 활동 재진입 처리 (결과 다시 보기 or 재도전)
- [ ] 네트워크 없을 때 안내 화면
- [ ] 화면 별 뒤로가기 버튼 동작 정리
- [ ] 태블릿(768px) 레이아웃 보정
- [ ] 텍스트·버튼 크기 접근성 점검
- [ ] 전체 플로우 시나리오 테스트 (처음~끝 3회 완주)

### 완료 기준
> 수업 중 예상되는 상황(인터넷 끊김, 뒤로가기, 중간에 껐다 켜기)에서 앱이 멈추지 않으면 완료.

## v1.0.0 : 완성 및 Netlify 배포

**목표**: 학생들이 실제로 URL로 접속해서 쓸 수 있는 완성 버전.

### 포함 기능
- [ ] Netlify 환경변수 `ANTHROPIC_API_KEY` 설정 및 보안 확인
- [ ] Netlify 배포 (index.html 업로드)
- [ ] 배포 URL로 실제 모바일 접속 테스트
- [ ] 갤럭시 탭·아이폰·안드로이드 폰 크로스 테스트
- [ ] 최종 디자인 polish (여백, 폰트 크기, 색상 미세 조정)
- [ ] 앱 파비콘 추가
- [ ] 수업 투입 전 교사 시뮬레이션 테스트

### 완료 기준
> 학생이 URL 하나로 접속해서 처음부터 끝까지 완주 가능. 교사가 옆에서 봤을 때 수업에 쓸 수 있겠다 싶으면 v1.0.0.
