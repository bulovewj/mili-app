# 작업 지시서 — v0.2c
# Screen 7: 뉴스 비교 (편향성 분석)

---

## 이 파일의 작업 범위

- **수정 대상**: `index.html` 내 `screen-news` 섹션
- **작업 내용**: 뉴스 앱 UI를 재현한 편향성 분석 시뮬레이션 화면 구현
- **건드리지 말 것**: 다른 화면, CSS 변수, localStorage 키, 라우팅 구조

---

## 화면 구성 개요

```
[1단계] 미션 안내 배너
        ↓
[2단계] 두 기사 카드 (탭 전환 또는 나란히)
        ↓
[3단계] 서술형 입력 + 제출
        ↓
[4단계] 더미 채점 결과 (screen-result)
```

---

## 상단 미션 배너

```
┌─────────────────────────────────┐
│ 📰 뉴스 비교                    │  ← 활동명 (--color-news 배경)
│ 같은 사건, 다른 시각!            │
│ 두 기사를 읽고 차이점을          │
│ 찾아봐! 🧐                      │
└─────────────────────────────────┘
```

---

## 두 기사 표시 방식

**탭 전환 방식** (모바일에서 나란히 보기 불편하므로)

```
┌─────────────────────────────────┐
│  [ 기사 A ]  |  [ 기사 B ]      │  ← 탭 버튼 2개
│  ▔▔▔▔▔▔▔▔▔▔▔                   │    선택된 탭 밑줄 + 굵게
└─────────────────────────────────┘
```

탭 A 선택 시 기사 A 표시, 탭 B 선택 시 기사 B 표시.  
기본값은 탭 A.

## 뉴스 기사 UI 재현

실제 뉴스 앱처럼 보이도록 구현.

### 기사 A

```
┌─────────────────────────────────┐
│ 🔵 건강한급식신문                │  ← 매체명 (파란 점 + 이름)
│                                 │
│ ┌───────────────────────────┐  │
│ │  🥗  채소 일러스트         │  │  ← 기사 이미지 영역 (CSS, 초록계열)
│ │  (초록 배경, 이모지)        │  │     height: 160px
│ └───────────────────────────┘  │
│                                 │
│ 학교 급식에 채식 선택제 도입,    │  ← 헤드라인 (font-weight: 900)
│ 학생 건강과 환경 모두 잡는다    │
│                                 │
│ ─────────────────────────────  │  ← 구분선
│                                 │
│ 전문가들은 채식 급식이 건강에   │  ← 본문 요약 (2~3줄)
│ 유익하며 탄소 배출 감소에       │
│ 기여한다고 밝혔다. 시범학교     │
│ 학생들의 만족도는 78%로        │
│ 높게 나타났다.                  │
│                                 │
│ 2025.03.15  |  교육팀 기자      │  ← 날짜·기자 (회색 소문자)
└─────────────────────────────────┘
```

### 기사 B

```
┌─────────────────────────────────┐
│ 🔴 학부모뉴스                    │  ← 매체명 (빨간 점 + 이름)

│ ┌───────────────────────────┐  │
│ │  😤  불만 아이 일러스트    │  │  ← 기사 이미지 영역 (CSS, 주황계열)

│ └───────────────────────────┘  │
│                                 │

│                                 │

│ 줄었다"며 불만을 토로했다.      │
│ 학부모들도 영양 불균형을         │
│ 우려하는 목소리가 높아지고      │
│ 있다.                           │
│                                 │
```

## "두 기사 모두 읽었어?" 확인 장치

탭 B를 한 번이라도 클릭해야 서술형 입력창이 나타남.

```javascript
let readBoth = false;

tabB.addEventListener('click', () => {
  readBoth = true;
  answerSection.style.display = 'block'; // 입력창 표시
});
```

입력창 표시 전:

```
┌─────────────────────────────────┐
│  💡 두 기사를 모두 읽은 후에    │
│     질문이 나타나요!             │
│     기사 B 탭도 눌러봐! 👆      │
└─────────────────────────────────┘
```

## 서술형 입력 영역

```
┌─────────────────────────────────┐
│ ✍️ 탐정 노트                    │
│                                 │
│ 두 기사는 어떻게 다르게         │
│ 보도하고 있어?                  │
│ 왜 그런 차이가 생겼을까?        │
│                                 │
│ ┌─────────────────────────────┐ │
│ │                             │ │  ← textarea
│ │  (최소 15자 이상 입력)       │ │     min-height: 120px
│ │                             │ │
│ └─────────────────────────────┘ │
│ 0 / 15자 이상                   │
│                                 │
│ [ 탐정 보고서 제출하기! 🕵️ ]    │
└─────────────────────────────────┘
```

최소 글자 수: **15자** (유튜브 탐정보다 약간 높게 설정)

## 데이터 저장

```javascript
const newsResult = {
  answer: studentText,
  score: 65,        // 더미 고정값
  grade: '중',      // 더미 고정값
  completed: true,
};
localStorage.setItem(STORAGE_KEYS.ACTIVITY_NEWS, JSON.stringify(newsResult));
// 저장 후 → screen-result로 이동
```
