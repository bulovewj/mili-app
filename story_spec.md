## 3. 스토리 흐름 전체 지도

```
[시작 화면] → [프롤로그 MAIN_STORY]
                   ├── Act 1: 교실 (classroom)
                   │     원일이 잘못된 정보를 퍼뜨림 → 친구들에게 신뢰 잃음
                   ├── Act 2: 하교길 (evening)
                   │     미리 등장 → 미디어 리터러시 필요성 안내
                   └── Act 3: 원일의 방 (room)
                         5개 미션 아이템 소개 → actionBtn: "첫 번째 미션 시작하기"
                              ↓
                    [DIAGNOSIS_BRIDGE_STORY]
                    미리: 진단 평가 안내 → actionBtn: "진단 평가 시작하기"
                              ↓
                    [진단 평가 screen-diagnosis]
                              ↓
                    [AFTER_DIAGNOSIS_STORY]
                    미리: 진단 완료 → actionBtn: "탐정 본부로 출발"
                              ↓
                    [screen-home (방 화면)] ←──────────────────┐
                              ↓                               │
              ┌──────────────────────────────┐                │
              │ 미션 클릭 (5개 중 선택)         │                │
              └──────────────────────────────┘                │
     각 미션 완료 후 → [AFTER_ACTIVITY_STORY.{미션명}]          │
     미리: 완료 메시지 → actionBtn: "탐정 본부로 돌아가기" ──────┘
                              ↓ (5개 모두 완료 시)
                    [FINAL_STORY]
                    미리: 수고했어 → actionBtn: "역량 지형도 보기"
                              ↓
                    [screen-radar (역량 지형도)]
```

## 9. 씬 오브젝트 필드 참조

### 일반 씬 필드

| 필드 | 타입 | 설명 |
|------|------|------|
| `bg` | string | 배경 ID (`classroom` / `evening` / `room`) |
| `left` | string | 왼쪽 캐릭터 ID |
| `right` | string | 오른쪽 캐릭터 ID |
| `speaker` | string | 현재 말하는 캐릭터 ID (자막창 이름 + 하이라이트 결정) |
| `text` | string | 대사 (`\n`으로 줄바꿈) |
| `label` | string? | 장소 전환 라벨 텍스트 (2초 표시 후 씬 렌더링) |
| `big` | boolean? | true이면 캐릭터를 센터 크게 표시 (miri 전용) |
| `special` | string? | 특수 연출 ID (`yt` / `kakao` / `room-items`) |
| `highlight` | string? | `room-items` special 사용 시 하이라이트할 아이템 ID |
| `actionBtn` | object? | `{ text: string, fn: string }` — 액션 버튼 표시 |

### 선택지 씬 필드 (type: 'choice')

| 필드 | 타입 | 설명 |
|------|------|------|
| `type` | `'choice'` | 선택지 씬 식별자 |
| `bg` | string | 배경 ID |
| `question` | string | 선택 질문 텍스트 |
| `options` | string[] | 선택지 배열 (2개) |
| `responses` | object[] | 각 선택지 선택 후 표시될 씬 객체 (일반 씬과 동일 구조) |

### 특수 연출 (special 필드 값)

| 값 | 렌더링 내용 |
|----|-----------|
| `yt` | 유튜브 카드: "🔴 YouTube · 조회수 98만회 / 교실 스마트폰 완전 허용?! / 클릭베이트 유튜버 · 조회수 989,432" |
| `kakao` | 카카오톡 카드: "💬 6학년 1반 단톡방 / 🚨 속보 / 내일 학교 안 가도 됨! / 오후 9:15" |
| `room-items` | 5개 미션 아이템 버튼 목록 (🖥️유튜브 / 📺뉴스 / 📱광고 / 💬AI / 💻카드뉴스) |
