## SNS 피드 UI 재현

인스타그램/SNS 스타일 피드 카드 3개 세로 목록.  
선택 가능: 복수 선택 허용 (체크박스 스타일)

## 중요 안내 (루브릭 반영)

> ⚠️ 이 활동의 목표는 **정확한 식별이 아닙니다**.  
> "의심하는 태도 + 왜곡 유형 언어화"가 평가 기준입니다.  
> 활동 안내 문구에 반드시 아래 내용 포함:

```
💡 탐정 팁!
"가짜인지 확실히 모르겠어도 괜찮아!
왜 의심스러운지 그 이유를 써주면 돼 😊"
```

```
왜 의심스러워? 🤔

(최소 10자)
```
