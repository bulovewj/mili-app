## 실제 문항 내용 (하드코딩)

### Q1 — 클릭베이트 식별

```
다음 유튜브 영상 제목 중 클릭베이트일 가능성이 가장 높은 것은?

① 과학자들이 발견한 새로운 공룡 화석 이야기
② 이걸 보면 절대 안 믿겠지만...충격 실화 (100% 사실!!!)
③ 우리나라 전통 놀이 종류와 방법 소개
④ 태양계 행성의 크기를 비교해봤어요

정답: ②
```

### Q2 — 편향성 인식

```
같은 사건을 다룬 두 기사가 서로 다른 내용을 강조하고 있다면,
그 이유로 가장 적절한 것은?

① 기자가 실수로 잘못 썼기 때문에
② 기사를 쓴 매체마다 관점과 입장이 다를 수 있기 때문에
③ 한쪽 기사는 반드시 거짓이기 때문에
④ 같은 사건이 두 번 일어났기 때문에

정답: ②
```

### Q3 — 광고·협찬 식별

```
유튜버가 영상에서 "이 영상은 ○○브랜드 협찬을 받았습니다"라고 밝혔어.
이 영상을 볼 때 어떤 태도가 가장 좋을까?

① 협찬을 받았으니 영상 내용이 모두 거짓이다
② 협찬 사실과 관계없이 내용을 그대로 믿는다
③ 협찬을 받았으므로 제품에 유리하게 만들어졌을 수 있다고 생각하며 본다
④ 협찬 영상은 절대 보지 않는 것이 좋다

정답: ③
```

### Q4 — 허위정보 판별

```
친구가 단체 채팅방에 "○○ 먹으면 감기 낫는다!"는 메시지를 보냈어.
가장 올바른 행동은?

① 나도 빠르게 다른 친구들에게 전달한다
② 재미있으니까 그냥 웃고 넘긴다
③ 공식 기관(질병청, 뉴스 등)에서 확인한 후 판단한다
④ 친구를 믿으니까 사실이라고 생각한다

정답: ③
```

## 채점 로직

```javascript
// 정답 배열 (1번 인덱스 = Q1)
const DIAGNOSIS_ANSWERS = [1, 1, 2, 2]; // 0-index: ②=1, ②=1, ③=2, ③=2

function calculateDiagnosisGrade(score) {
  if (score === 4) return { grade: '상', level: '수석 탐정', emoji: '🏆' };
  if (score >= 2) return { grade: '중', level: '탐정', emoji: '🕵️' };
  return { grade: '하', level: '탐정 견습생', emoji: '🔰' };
}

// localStorage 저장
localStorage.setItem(STORAGE_KEYS.DIAGNOSIS_SCORE, score);
localStorage.setItem(STORAGE_KEYS.DIAGNOSIS_GRADE, grade);
```
