---

## 이 파일의 목적

기존 index.html에 임의로 들어있던 콘텐츠를 **교사와 함께 설계한 실제 교육 콘텐츠**로 교체한다.
기존 상/중/하 루브릭 채점을 **P/F 채점 시스템**으로 교체한다.

## 1. 진단 퀴즈 (Module 01)

### 변경 사항
- 기존: 단순 식별형 ("다음 중 클릭베이트는?")
- 변경: 개념·태도·행동 판단형 ("~한 방법 중 옳지 않은 것은?")
- **정답 후 개념 설명 박스 추가** (Q2~Q4)
- **P/F 없음** — 수준 파악용으로만 사용

### 문항 순서
허위정보(Q1) → 클릭베이트(Q2) → 편향성(Q3) → 광고·협찬(Q4)

---

### Q1 — 허위정보 (방법 판단형 / 옳지 않은 것)

```
인터넷에서 본 정보가 사실인지 확인하는 방법으로 옳지 않은 것은?

① 뉴스 기사나 공식 기관 사이트에서 같은 내용을 찾아본다
② 공유된 횟수가 많을수록 믿을 수 있다고 생각한다  ← 정답
③ 글을 쓴 사람이 누구인지 확인한다
④ 여러 곳에서 같은 내용을 다루는지 비교해본다
```

**정답 피드백**: "✅ 맞아! 공유 횟수가 많다고 해서 사실인 건 아니야. 가짜 정보도 빠르게 퍼질 수 있어!"
**오답 피드백**: "❌ 다시 생각해봐! 공유 횟수가 많다고 사실인 건 아니야."
**개념 설명 없음** (Q1은 설명 박스 미표시)

### Q2 — 클릭베이트 (행동 판단형 / 올바른 것)

```
유튜브에서 "이걸 보면 인생이 바뀜!! (100% 실화)"라는 제목의 영상을 
발견했어. 이때 가장 올바른 행동은?

① 제목이 자극적이니까 분명 재밌는 내용일 거라 생각한다
② 조회수가 높으니까 믿을 수 있다고 판단한다
③ 제목만 보고 바로 친구에게 공유한다
④ 제목이 과장됐을 수 있으니 무조건 믿지 않는다  ← 정답
```

**정답 피드백**: "✅ 정답이야! 자극적인 제목일수록 한 번 더 의심해보는 습관이 중요해!"
**오답 피드백**: "❌ 다시 생각해봐! 자극적인 제목이라고 무조건 좋은 내용은 아니야."

**💡 개념 설명 박스**
```
이런 제목을 '클릭베이트'라고 해요!

클릭베이트란 사람들이 궁금해서 클릭하도록 제목을 일부러 자극적이거나
과장되게 만드는 방법이야. "충격", "절대", "100% 실화" 같은 표현이 자주 쓰여.
제목만 보고 판단하지 말고, 실제 내용을 확인하는 습관을 갖자!
```

---

### Q3 — 편향성 (방법 판단형 / 옳지 않은 것)

```
뉴스를 비판적으로 읽는 방법으로 옳지 않은 것은?

① 기사를 쓴 매체가 어디인지 확인한다
② 제목과 본문 내용이 일치하는지 살펴본다
③ 구독자가 많은 유튜버가 말한 내용이면 믿어도 된다  ← 정답
④ 같은 사건을 다른 매체에서도 어떻게 다루는지 비교해본다
```

**정답 피드백**: "✅ 맞아! 구독자 수가 많다고 정보가 정확한 건 아니야!"
**오답 피드백**: "❌ 다시 생각해봐! 구독자 수는 정보의 정확성과 관계없어."

**💡 개념 설명 박스**
```
이걸 '편향성'이라고 해요!

편향성이란 한쪽 방향으로 치우쳐서 정보를 전달하는 것을 말해.
구독자 수나 유명함이 정보의 정확성을 보장하지 않아.
뉴스를 읽을 때는 누가 썼는지, 어떤 관점인지,
다른 매체는 어떻게 보도하는지 비교하는 습관이 중요해!
```

### Q4 — 광고·협찬 (행동 판단형 / 올바른 것)

```
좋아하는 유튜버가 영상에서 특정 제품을 강력 추천하고 있어.
이때 가장 올바른 행동은?

① 좋아하는 유튜버니까 추천 제품을 바로 구매한다
② 협찬 표시가 없으면 유튜버의 진짜 의견이라 믿는다
③ 협찬이나 광고 여부를 확인하고 내용을 비판적으로 본다  ← 정답
④ 광고가 포함된 영상은 무조건 보지 않는다
```

**정답 피드백**: "✅ 정답! 협찬 여부를 먼저 확인하고 비판적으로 보는 게 중요해!"
**오답 피드백**: "❌ 다시 생각해봐! 유튜버가 좋아도 광고 여부는 항상 확인해야 해."

**💡 개념 설명 박스**
```
이걸 '광고·협찬'이라고 해요!

협찬이란 기업이 유튜버나 인플루언서에게 돈이나 제품을 제공하고
홍보를 부탁하는 것을 말해. 협찬을 받은 콘텐츠는 제품에 유리하게
만들어질 수 있어. 표시가 있든 없든 항상 "이 사람이 협찬을 받은 건
아닐까?" 라고 한 번쯤 생각해보는 습관이 중요해!
```

## 3. 뉴스 비교 (Screen 7)

### 주제
AI 검색 vs 도서관 이용 (독서 습관 변화)

### 기사 콘텐츠

**기사 A — 디지털교육신문 (AI 검색 긍정)**
```
매체명: 🔵 디지털교육신문
헤드라인: "AI 검색으로 궁금증 바로 해결, 학생 학습 효율 높아졌다"

본문:
한국교육개발원이 발표한 설문 결과에 따르면 초등학생의 74%가
"AI 검색 덕분에 공부할 때 막히는 부분을 빠르게 해결할 수 있다"고
응답했다. 교사들도 "학생들이 스스로 궁금증을 해결하려는 시도가
늘었다"고 밝혔다. AI 검색은 이제 학생들에게 없어서는 안 될
학습 도구가 되었으며, 이를 적극 활용하도록 돕는 것이
교육의 방향이 되어야 한다.     ← 기자 의견 문장

날짜: 2025.05.12 | 교육팀 기자
```

**기사 B — 책사랑뉴스 (도서관·독서 옹호)**
```
매체명: 🔴 책사랑뉴스
헤드라인: "AI만 찾는 아이들, 도서관 텅텅… 깊이 있는 사고력 잃어가"

본문:
전국 초등학교 도서관 이용률이 3년 새 41% 감소한 것으로 나타났다.
한 초등학교 사서 교사는 "예전엔 쉬는 시간마다 아이들로 북적였는데
요즘은 하루 종일 손님이 없는 날도 있다"고 말했다.
빠른 답을 주는 AI 검색에 익숙해진 아이들이 책을 통해 스스로
생각하는 힘을 잃어가고 있는 것은 아닌지 깊이 우려된다.  ← 기자 의견 문장

날짜: 2025.05.12 | 사회팀 기자
```

```javascript
욕설, 개인정보, 혐오표현이 포함된 경우 채점 거부하고 재입력 안내.

반드시 아래 JSON 형식으로만 응답해:
{
  "keyword_count": 0~4 (포함된 핵심 내용 개수),
  "score": 0 | 40 | 70 | 100,
  "pass": true | false,
  "feedback": "학생에게 전달할 피드백 (2문장, 친근한 초등학생 말투)",
  "found_keywords": ["인정된 내용 1", "인정된 내용 2"]
}
`;
```

### 활동별 최종 점수 계산

```javascript
function calculateFinalScore(activityKey, scores) {
  const weights = {
    youtube:  { mc: 0.30, essay: 0.70, essayParts: { essay1: 1.0 } },
    news:     { mc: 0.20, essay: 0.80, essayParts: { essay1: 0.50, essay2: 0.50 } },
    ad:       { mc: 0.40, essay: 0.60, essayParts: { essay1: 1.0 } },
    ai:       { mc: 0.30, essay: 0.70, essayParts: { essay1: 1.0 } },
    cardnews: { mc: 0.30, essay: 0.70, essayParts: { essay1: 0.40, essay2: 0.60 } },
  };

  const w = weights[activityKey];
  const mcScore = scores.mc * w.mc;
  let essayScore = 0;
  for (const [part, weight] of Object.entries(w.essayParts)) {
    essayScore += (scores[part] || 0) * weight * w.essay;
  }
  return Math.round(mcScore + essayScore);
}

// P/F 판정
function getPassFail(finalScore) {
  return finalScore >= 50 ? 'P' : 'F';
}
```
