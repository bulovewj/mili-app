## 이 버전의 목표

v0.1에서 만든 뼈대(index.html)에 **실제 콘텐츠를 채워 넣는 단계**입니다.  
API 없이도 모든 화면을 처음부터 끝까지 체험할 수 있는 상태가 목표입니다.

## v0.2 완료 기준

- [ ] 진단 4문항이 실제 내용으로 채워지고 자동 채점됨
- [ ] 5개 시뮬레이션 화면이 실제 미디어 UI처럼 보임
- [ ] 각 활동에서 서술형 답변 입력 후 제출 가능
- [ ] 제출 후 더미 채점 결과 화면으로 이동 (AI 없이 고정값)
- [ ] 3개 이상 완료 시 역량 지형도(레이더 차트) 더미 데이터로 표시
- [ ] API 키 없이도 앱 전체 플로우 완주 가능
