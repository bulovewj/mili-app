# 작업 지시서 — v0.2b
# Screen 6: 유튜브 탐정 (클릭베이트 식별)

---

## 이 파일의 작업 범위

- **수정 대상**: `index.html` 내 `screen-youtube` 섹션
- **작업 내용**: 유튜브 앱 UI를 재현한 클릭베이트 탐지 시뮬레이션 화면 구현
- **건드리지 말 것**: 다른 화면, CSS 변수, localStorage 키, 라우팅 구조

---

## 화면 구성 개요

이 화면은 **2단계**로 구성됩니다.

```
[1단계] 유튜브 UI 재현 + 썸네일 선택
        ↓ 썸네일 선택 + 이유 입력 + 제출
[2단계] 더미 채점 결과 (screen-result로 이동)
```

---

## 1단계 — 유튜브 탐정 메인 화면

### 썸네일 선택 인터랙션

```javascript
// 썸네일 탭 시:
// 1. 선택한 썸네일에 굵은 테두리 표시 (--color-youtube, 3px)
// 2. 우측 상단에 ✓ 뱃지 표시
// 3. 다른 썸네일 선택 시 이전 선택 해제
// 4. 선택된 썸네일 ID를 변수에 저장 ('A', 'B', 'C')
```

### 서술형 이유 입력

```
┌─────────────────────────────────┐
│ 왜 클릭베이트라고 생각했어?      │
│ 이유를 써봐! ✍️                  │
│                                 │
│ ┌─────────────────────────────┐ │
│ │                             │ │  ← textarea
│ │  (최소 10자 이상 입력)       │ │     min-height: 100px
│ │                             │ │     font-size: 16px
│ └─────────────────────────────┘ │
│ 0 / 10자 이상                   │  ← 글자 수 카운터
│                                 │
│ [ 탐정 보고서 제출하기! 🕵️ ]    │  ← 썸네일 선택 AND 10자 이상일 때 활성화
└─────────────────────────────────┘
```

```javascript
// 글자 수 실시간 카운트
// 조건: 썸네일 선택됨 AND 글자 수 >= 10
// → 버튼 활성화 (--color-primary 배경)
// 조건 미충족 → 버튼 비활성화 (회색, cursor: not-allowed)
```

## 썸네일 CSS 구현 명세

이미지 파일 사용 금지. 아래처럼 CSS + SVG로 구현.

```css
/* 썸네일 A: 우주/행성 느낌 */
.thumb-a {
  background: linear-gradient(135deg, #1a1a2e 0%, #16213e 50%, #0f3460 100%);
  /* 별 패턴: CSS radial-gradient dots */
  height: 180px;
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 48px;  /* 🪐 이모지 크게 */
}

/* 썸네일 B: 클릭베이트 - 시선을 잡는 강렬한 배경 */
.thumb-b {
  background: linear-gradient(135deg, #ff0000 0%, #ff6600 100%);
  height: 180px;
  border-radius: 8px;
  /* 느낌표 패턴 텍스트 반복: CSS content or span */
  position: relative;
  overflow: hidden;
}
/* 느낌표 배경 패턴 (pseudo-element) */
.thumb-b::before {
  content: '❗❗❗❗❗❗❗❗❗❗❗❗❗❗❗❗';
  position: absolute;
  font-size: 24px;
```

## 데이터 저장

```javascript
// 제출 버튼 클릭 시 저장
const youtubeResult = {
  selectedThumb: 'B',        // 학생이 선택한 썸네일
  answer: studentText,       // 학생 서술형 답변
  score: 65,                 // 더미 고정값 (v0.3에서 AI로 교체)
  grade: '중',               // 더미 고정값
  completed: true,
};
localStorage.setItem(STORAGE_KEYS.ACTIVITY_YOUTUBE, JSON.stringify(youtubeResult));

// 저장 후 → screen-result로 이동
// screen-result에 활동명, 더미 등급, 더미 피드백 전달
```
